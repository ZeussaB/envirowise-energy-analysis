# Energy Consumption Data Science Project

This project demonstrates the use of data science techniques to analyse and predict
electricity consumption using open government energy data.

## Tools Used
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Jupyter Notebook

## Project Structure
- `data/` – Contains the energy dataset
- `energy_consumption_analysis.ipynb` – Main analysis and model implementation
- `requirements.txt` – Project dependencies

## Purpose
The project supports environmental sustainability by providing data-driven insights
to reduce energy consumption in commercial buildings.
